CSS - Cascading Style Sheet

Dove HTML descrive la struttura di un certo contenuto, i CSS sono pensati per dare vesta grafica dignitosa a un contenuto che altrimenti sarebbe minimale.

Tipologie di utilizzo di CSS

Possono essere utilizzati in tre modi diversi:

  1. Inline: sono inclusi nel tag che si vuole modificare, utilizzando l'attributo style
  2. Incorporati: sono inclusi nella sezione head della pagina sottoforma di regole CSS all'interno del tag style
  3. Esterni: vengono inclusi come file CSS, all'interno dell'head utilizzando il tag link

Dall'alto verso il basso sono più specifici, cioè un CSS inline ha la precedenza su uno incorporato, che a sua volta ha la precedenza su uno esterno.

Viceversa i fogli esterno permettono omogeneità alla pagina su tutto il sito, quelli incorporati su tutta la pagina, quelli inline solo dove sono applicati

Come sono fatti

I fogli di stile sono insiemi di regole, e ogni regola è del tipo 

nome:valore;
e sono applicate su elementi selezionati attraverso un selettore

Esempio:

            p {
                font-size: 12pt;
                color: blue;
            }

Questo è un paragrafo il cui font è stato modificato tramite una regola inline.

Selezionare classi e id

A volte può essere necessario andare a individuare una qualche sezione in modo particolare, allora possono essere utilizzati gli attributi class (per individuare tutti gli elementi che condividono qualcosa) e id (per individuare un solo elemento).
