Come fare il deployment di un oggetto da Phpstorm.

  1. Andare nei Tools -> Deployment -> Configuration...
  2. Premere il pulsante con il simbolo + per aggiungere una nuova configurazione di deploy
  3. Dare un nome significativo alla configurazione creata
  4. Selezionare come Type SFTP
  5. Se non già presente, aggiungere una configurazione SSH, selezionando i tre puntini
  6. Nel campo Host inserire l'indirizzo di Winnie (IP 10.4.55.123 porta 22)
  7. Nel campo Username inserire lo username dato all'inizio dell'anno
  8. Come Authentication type, lasciare password
  9. Provare con test connection se va tutto bene, altrimenti modificare
  10. Premere OK e tornare alla schermata precedente
  11. Impostare la Root Path del progetto selezionando l'icona della cartella e mandando nella propria cartella, che si trova all'indirizzo home/~corra/public_html/sottocartella
  12. In WebServer URL inserire l'URL del proprio progetto su Winnie, ad esempio http://10.4.55.123/~corra/introduzione/deployment.html
  13. Spostarsi sul tag Mappings
  14. LocalPath lasciare quello registrato
  15. Negli altri due campi, a meno di cose particolari, inserire / e /
  16. Premere infine OK e si avrà una configurazione di deployment
  17. Per comodità nel menù tools -> deployment si può spuntare Automatic upload
